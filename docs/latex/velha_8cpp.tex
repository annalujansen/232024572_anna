Verifica o estado atual do jogo da velha. 

Esta função analisa um tabuleiro 3x3 e determina\+:
\begin{DoxyItemize}
\item Se houve um vencedor (X ou O),
\item Se houve empate,
\item Se o jogo ainda está em andamento,
\item Ou se a configuração do tabuleiro é impossível.
\end{DoxyItemize}
